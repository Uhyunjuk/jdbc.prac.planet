## :computer: 주제
오라클에 태양계 행성 테이블 만들고 데이터 삽입 후 이클립스와 연동해서 콘솔에 출력해보기

## :pushpin: 요구사항
1. 태양계 행성 전체 조회
2. 행성 이름을 입력 받아서 관련 데이터를 조회
3. 설명을 듣고 싶은 행성의 이름을 입력받아서 행성의 특징 출력
4. 나만의 행성 추가
5. 선택한 행성 삭제
6. 프로그램 종료
